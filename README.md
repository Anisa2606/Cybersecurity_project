# 🔐 AI-Based Network Intrusion Detection System

## 📌 Overview
This project presents an AI-powered Network Intrusion Detection System (IDS) designed to classify network traffic as **benign** or **malicious** using machine learning techniques. The system analyzes network traffic patterns from the CIC-IDS-2018 dataset to identify potential cyber threats and improve network security.

The project aims to automate threat detection and reduce false positives using data preprocessing, feature engineering, and machine learning classification algorithms.

---

## 🚀 Features
- Detection of malicious and benign network traffic
- Machine learning-based threat classification
- Data preprocessing and feature engineering
- High detection accuracy with reduced false positives
- Performance evaluation using multiple metrics
- Scalable and efficient intrusion detection pipeline

## 📂 Dataset
The project uses the **CIC-IDS-2018 Dataset**, which contains realistic network traffic data including both normal activity and cyber attacks.

### Attack Categories Included:
- DDoS Attacks
- Brute Force Attacks
- Botnet Activity
- Web Attacks
- Infiltration Attacks

🔗 Dataset Link:  
https://www.unb.ca/cic/datasets/ids-2018.html
